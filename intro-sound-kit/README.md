# Intro Sound Kit

Converted from `/Users/cypher/Downloads/glass_animals` for Cardputer MPC.

All WAVs in `samples/` are:

- PCM WAV
- mono
- 16-bit
- 16000 Hz
- trimmed to short one-shots for the Cardputer's no-PSRAM sample pool

To use on the SD card, copy the WAVs to:

```text
/cardputer-mpc/samples/intro/
```

Copy `kits/intro.json` to:

```text
/cardputer-mpc/kits/intro.json
```
